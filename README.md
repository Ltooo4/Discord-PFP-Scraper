## Installation

Installing Discord Pfp Scraper

```bash
  git clone https://github.com/Ltooo4/Discord-PFP-Scraper.git
  cd Discord-PFP-Scraper
  pip install -r requirements.txt
  python main.py
```
Put your discord token in config.json
